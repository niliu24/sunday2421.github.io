1. 创建 QWidget 程序

执行 xmake create -t qt.widgetapp 创建一个 qt 模板工程，工程目录如下:

  • QtProject
    • src
      • main.cpp
      • mainwindow.h
      • mainwindow.cpp
      • mianwindow.ui
    • xmake.lua

如果系统中已经安装了 qt 环境，执行 xmake b && xmake r 即可编译运行，程序将显示一个空白窗口。
c++代码与 qtcreator 生成的模板工程一样，xmake.lua 内容如下:

与非 qt 工程内容基本相同，主要差异在增加了 add_rules("qt.widgetapp") 来实现 qt 工程编译。
使用 qtdesigner 编辑 ui 文件，使用 uic mainwindow.ui -o ui_mainwindow.h 将 ui 文件转化为 c++ 头文件。
