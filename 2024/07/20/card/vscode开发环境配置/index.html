配置c/c++开发环境

使用 clangd 作为 lsp

  • vscode 插件

    • clangd: 代码补全和静态检查
    • clang-format：代码格式化

  • 解决 clangd 找不到当前项目子目录下头文件的问题

    1. 在 settings.json 文件中增加如下配置

    2. 或者使用 cmake/xmake 等工具生成 compile_commands.json 文件

配置ssh服务

  • 设置 vscoed 免密码登录
    • 将 .ssh\id_rsa.pub 文件复制并改名为 _authorized_keys_，然后拷贝到服务端的 ~/.ssh/ 目录下
  • 服务端重装后，vscode 连接相同 ip 出现 "Host key verification failed" 错误，重装系统后 ssh 签名不一致，在本地主机上执行 ssh-keygen -R [server_ip] 后重新设置连接

设置代码片段

官方参考文档 Snippets in Visual Studio Code

  1. ctrl+shift+p 调出命令控制台，输入 snippet 选择代码片段：配置用户代码片段，然后选择对应的编码语言将打开对应的配置文件

  2. 参考配置文件中的注释，编写自定义片段即可

Code Runner插件配置

  • vscode 的单文件代码运行插件，支持 c/c++、python、lua 等代码一键运行，实质上还是通过命令行来执行代码，可在 settings.json 文件中进行自定义

  • 支持的内置变量

    • $dir: 执行文件所在目录
    • $fileName: 执行文件名
    • $fileNameWisthoutExt: 执行文件名去掉类型后缀
