git子模块管理

Yang Jian
2024-07-20
card
git

添加子模块

  1. git submodule add repo_url [path] 从 url 添加子模块到 path
  2. 添加子模块成功后，会在 .gitmodules 添加子模块的配置信息
  3. git add .gitmudules path 提交子模块修改
  4. git submodule update --init --recursive 更新和拉取子模块代码
  • 拉取包含子模块的项目，初始化和更新子模块 git clone --recursive repo_url

删除子模块

  1. 删除子模块注册 git submodule deinit -f submodule_path
  2. 删除子模块相关元数据 rm -rf .git/modules/submodule_path
  3. 删除子模块索引 git rm -rf --cached submodule_path
  4. 编辑 .gitmodules 文件，删除子模块相关配置
