linux 上搭建 c/c++ 开发环境

使用 gcc 工具链编译程序

  1. 安装工具链: sudo apt install build-essential

  2. 检查 gcc/g++ 是否安装成功: gcc --version, g++ --version；gcc 用于编译 c 代码，g++ 用于编译 c++ 代码

  3. 使用 vim 或者其他编辑器编写 hello-world 程序

  4. 编译程序: g++ hello.cpp -o hello；执行成功后将生成名为 hello 的可执行程序

  5. 终端中执行 ./hello 运行程序

使用 clang 作为编译工具
  1. 安装相关工具: sudo apt install clang clangd clang-format clang-tidy
问题记录

[!question] 在 linux mint 上使用 clang 作为编译工具，出现 /usr/bin/ld: 找不到 -lstdc++ 错误
原因是 libstdc++ 在 /usr/lib/x86_64-linux-gnu 目录下，名称为 libstdc++.so.6，名称不对，所以 ld 找不到，建立符号链接即可: sudo ln -s /usr/lib/x86_64-linux-gnu/libstdc++.so.6 /usr/lib/x86_64-linux-gnu/libstdc++.so
