    linux 环境上安装 nodejs

    • ubuntu 上可以使用 apt 进行安装，但是版本比较低
    • 还可以使用 nvm 进行 nodejs 环境的安装和版本管理

    使用 nvm 管理 nodejs

    1. 使用如下命令安装 nvm

    2. 安装成功后，终端打印消息末尾将输出如下内容，将其写入到终端配置文件中，如 ~/.bashrc 文件

    3. 使用 source 命令更新配置文件后，使用 nvm ls-remote 命令即可查看到远程仓库中所有的 nodejs 版本信息

    4. 选择一个最新的长期支持版本进行安装，使用 nvm install v20.11.1 命令即可安装指定版本

    5. 因为网络原因使用 npm 安装包可能很慢，可以使用 npm install -g cnpm 安装 cnpm，然后使用 cnpm 替换 npm 来安装 nodejs 包即可
