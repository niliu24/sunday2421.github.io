    • QML Book—基本元素

    元素可以分为可视化元素和非可视化元素。

    • 可视化元素有着几何形状并可以在屏幕上显示
    • 非可视化元素提供常用功能，通常用于操作可视化元素

    基础元素对象(Item Element)

    Item 是所有可视化元素的基础对象，所有其它可视化元素都继承自 Item 。它本身不会有任何绘制操作，但是定义了所有可视化元素共有的属性：

    分组 属性(Properties)
    几何属性(Geometry) x,y：定义元素左上角的位置
    width,hieght：定义元素的显示范围
    z：堆叠次序，定义元素之间的重叠顺序
    布局操作(Layout) anchors (锚定)，包括 left/right/top/bottom/vertical center/horizontal center 等子属性，与 margins (间距)一起定义元素与其它元素之间的位置关系
    按键操作(Key) 附加属性 key 和 keyNavigation 来控制按键操作，focus 处理输入焦点操作
    转换(Transformation) scale 和 rotate 转换
    transform 用于通用的 x/y/z 属性转换
    transformOrigin 设置旋转基点
    可视化(Visual) opacity (不透明度)：设置透明度
    visible ：控制元素是否显示
    clip ：用来限制元素边界的绘制
    smooth ：用来提高渲染质量
    状态定义(State) states ：提供元素当前所支持的状态列表
    transition ：定义状态转变动画

    [!info] Item 元素通常被作为其它元素的容器使用，类似 HTML 语言中的 div 元素。

    矩形框元素(Rectangle)

    Rectangle 是 Item 对象的一个扩展，增加了一个颜色来填充。

    • 支持边界的定义，使用 border.color, border.width 定义边界
    • 可以使用 radius 属性来创建一个圆角矩形

    文本元素(Text)

    显示文本需要使用 Text ，该元素只显示文本，不会渲染背景；该元素会根据给定的 text 属性值和字体自动计算初始化的宽高：

    • horizontalAlignment 和 verticalAlignment 属性用来设置对齐效果
    • style 和 styleColor 属性来配置文字的外框效果、浮雕效果或者凹陷效果，来提升文文本的渲染效果
    • elide 属性允许设置省略号位置，当文本太长时用省略号替代部分文本
    • wrapMode 属性用来设备文本换行，仅在明确设置了宽度属性时才生效

    图像元素(Image)

    • source 属性指定图像文件的链接信息，可以使用网络链接；或者将文件放入到 qrc 文件中管理，再通过文件名引用
    • PreserveAspectCrop 可以避免剪裁图像数据被渲染到图像边界外
    • clip 用来约束边界矩形的绘制

    鼠标区域元素(MouseArea)

    这是一个矩形的非可视化元素对象，可以通过它来捕捉鼠标事件。
