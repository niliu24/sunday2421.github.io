windows 端桌面开发环境搭建

  1. 下载在线安装程序: qt在线安装程序下载地址

    1. 进入 4.6 目录

    2. 选择对应平台的安装程序，如 windows 平台选择 qt-unified-windows-x64-4.6.1-online.exe

    3. 下载页面如图所示:

  2. 安装程序

    1. 执行 qt-unified-windows-x64-4.6.1-online.exe 程序，将启动界面安装程序:

    2. 若没有账号则先选择 注册 ，然后再输入账号密码，选择 下一步 进入开源义务选择界面:

    3. 勾选 同意开源条款 和 个人用户 , 选择 下一步 进入欢迎界面，直接选择 下一步 进入 contribute 界面:

    4. 选择 help… 或者 Disable 均可，然后选择 下一步 ，进入安装目录选择界面:

    5. 选择一个合适的安装目录用于存放 qt 程序，注意目录中不要包含中文；然后勾选 Custom instalation ，选择 下一步 进入组件选择界面:

    6. 第一步先勾选 Archive , 然后点击 筛选 ，中间的组件选择窗口的 Qt 选项中将多出许多不同版本 qt 供选择，勾选 Qt 5.15.2 即可，再选择 下一步 进入许可协议界面

    7. 勾选同意协议，选择 下一步 快捷方式选择界面

    8. 直接 下一步 进入安装界面

    9. 选择 安装 等待安装完成即可

    嵌入式端开发环境搭建

  1. 下载源码: qt源码下载地址

    1. 进入 5.15 目录

    2. 进入 5.15.2 目录

    3. 进入 single 目录，即 Index of /archive/qt/5.15/5.15.2/single

    4. 选择 zip 或 tar.gz 格式下载均可

    5. 下载页面如图所示:

  2. 选择一个目录解压源码包: tar xvf qt-everywhere-src-5.15.2.tar.xz (如果是 zip 格式，使用 unzip 命令， unzip --help 查看帮助)

  3. cd qt-everywhere-src-5.15.2 进入解压后的源码目录

  4. ./configure --help 查看编译配置选项说明

  5. 执行 ./configure 配置 qt 编译选项

    • -prefix /home/linaro/Workspace/qtlib/: 指定 qt 安装目录，不指定则默认安装在 /usr/local/Qt-5.15.2
    • -opensource -confirm-license:  使用开源协议
    • -nomake tests -nomake examples: 不要编译测试用例和示例程序
    • -mp: 多线程编译
    • -release -optimize-size -strip: 编译为 release 版本，优化大小，去除符号表
    • -skip *: 不要编译该模块
      • loaction: 建议 skip 该模块，否则在 make install 时会报 cannot find -lclip2tri cannot find -lpoly2tri 的错误；如果一定要编译该模块，可以编译完后进入 src 目录，找到对应的组件目录，执行 make 手动编译该组件
      • qtvirtualkeyboard: 关闭 skip 该模块，否则在 make install 时会报 libqtopenwnn.a: No such file or directory 的错误；解决办法是进入 qtvirtualkeyboard 目录，执行 make 编译该组件
    • -qpa xcb: 编译 xcb 插件，否则运行目标程序时会出现 xcb 插件缺失的错误

  6. 执行 make -j6 编译源码

  7. 执行 make install 安装到指定目录，若采用默认目录在需要添加 sudo 权限
